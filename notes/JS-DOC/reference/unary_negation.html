Unary negation (-)

The unary negation operator (-) precedes its operand and negates it.

Syntax

Operator: -x

Examples

Negating numbers

const x = 3;
const y = -x;

// y = -3
// x = 3

Negating non-numbers

The unary negation operator can convert a non-number into a number.

const x = "4";
const y = -x;

// y = -4
