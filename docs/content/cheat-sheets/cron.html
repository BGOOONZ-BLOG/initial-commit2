Format

{: .-setup}

*    *    *    *    *  command to be executed
┬    ┬    ┬    ┬    ┬
│    │    │    │    └─  Weekday  (0=Sun .. 6=Sat)
│    │    │    └──────  Month    (1..12)
│    │    └───────────  Day      (1..31)
│    └────────────────  Hour     (0..23)
└─────────────────────  Minute   (0..59)

Operators

Operator Description

, separate individual values

Examples

Example Description
0 * * * * every hour
*/15 * * * * every 15 mins
